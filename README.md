# terradoc-bor: Repositório de Áudios Xavante

Este repositório contém arquivos de áudio em língua **Xavante**, utilizados em plataformas digitais, materiais educacionais, documentação linguística, descrição gramatical e pesquisas acadêmicas vinculadas ao projeto **Plataformas Indígenas**.

Os arquivos aqui reunidos podem servir como base para dicionários digitais, corpora linguísticos, materiais escolares, estudos fonéticos e fonológicos, reconhecimento automático de fala, síntese de fala e modelos de linguagem voltados a línguas indígenas brasileiras.

## Objetivo

O objetivo deste repositório é oferecer uma organização transparente, rastreável e citável dos arquivos de áudio Bororo usados em:

- plataformas digitais;
- dicionários digitais;
- materiais escolares;
- documentação linguística;
- descrição gramatical;
- estudos fonéticos e fonológicos;
- artigos acadêmicos;
- tecnologias de fala;
- modelos de linguagem para línguas de baixo recurso.
